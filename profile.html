Your Profile

These rules tell Accord how to fill Google Forms for you. When a form asks for "Phone Number", Accord finds the matching rule below and types in the value you saved here — so you never have to type it again.

How it works

  • Each rule represents one piece of personal info you reuse — phone, college, roll number, etc.
  • Pick a match type: contains is most forgiving, equals exactly is strictest.
  • List one or more patterns (comma-separated). Any pattern matches the question name → the rule fires.
  • The value is what Accord types into the field. For Name and Email, you can use your Google account info instead of a static value.
  • Toggle a rule off to skip it during autofill — handy if you want to leave one field blank for a particular form session.
